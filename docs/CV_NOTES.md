# CV Notes

## English

**Project title:** Binary Complementary Sequence Design and Validation

- Re-engineered a MATLAB/Windows-MEX signal-processing prototype into a
  cross-platform Python package for ZCP, GCS, and CZCS construction and
  verification.
- Implemented tested aperiodic auto/cross-correlation primitives, Kronecker
  ZCP construction, transition-pruned candidate generation, and reproducible
  small-length exhaustive searches.
- Built a regression pipeline that verified three reported ZCP constructions
  and audited 1,581 archived CZCS records, identifying 527 inconsistent
  historical results.
- Added command-line tooling, automated tests, CI configuration, mathematical
  documentation, and preserved legacy artifacts for research provenance.
