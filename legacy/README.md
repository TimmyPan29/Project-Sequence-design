# Legacy Research Artifacts

This directory preserves the original MATLAB, C, and Windows MEX material.
Files are intentionally not corrected in place because they are the provenance
for the refactored implementation and validation report.

## Contents

- `midterm/`: length-$37$/$41$ searches, MEX source, and Kronecker scripts;
- `final/`: GCP/CZCS search scripts and binary-only generalized MEX modules;
- `zcp_l41/`: the archived length-$41$ script and its recorded result.

Known execution and correctness boundaries are listed in
[`docs/VALIDATION_REPORT.md`](../docs/VALIDATION_REPORT.md).
